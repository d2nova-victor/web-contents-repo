Receptionist (live attendant)

You can select a receptionist to answer calls if you prefer a live attendant. When the receptionist is not available to answer an incoming call, you can set up other extensions or departments to receive the call.

  • Even then, you will need to set up lunch break or after work greetings so that callers during those times can leave a message for the receptionist.
  • Be sure to also set up a speed dial menu so calls can be forwarded to the right team when the receptionist is not available.
