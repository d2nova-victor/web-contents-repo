領導團隊

David Wong

董事長

David Wong 王爾仁博士有超過30年的豐富經驗在發展並經營通訊公司。事業的發展初期是從當語音壓縮演算法的研究員，是歷史上首次成功用 ARPANET 網路撥打電話的 ARPA 研發團隊成員。1974到1987年間在專業領域進行深耕，任職各樣科學研究、教授及技術管理工作等，於1987合夥創辦 Spectron Microsystems ，任為執行長，並研發出了世界上首個 DSP 操作系統，之後成為德州儀器 DSP BIOS 的基礎技術。王博士隨後在1993創立 D2 Technologies ，1997年協助創立 Connected System 公司，擁有語音訊息及 Unified Messaging ODM 的先驅技術。在2000年，D2 Technologies 被 Virata 併購，王博士隨後就任 GlobespanVirata (之後合併入 Conexant )的 VoIP 副總。
在2003年再度創立 D2 Technologies ，讓次世代的 IP 通訊技術成為現實。在2016年創立 D2 Nova 致力於開發 SaaS 雲端通訊。王博士同時也在多個新創科技公司擔任董事，其中包含 Trellisware Technologies(San Diego, USA) 、 Fanvil 方位通訊(中國深圳)及 FMSoft 飛漫軟件(中國北京)等，並且在通訊領域擁有多項專利及學位，包含加州大學聖塔芭芭拉分校的電機系學士、碩士及博士。

David Lindsay

技術長

David Lindsay 擁有超過30年的語音系統專業工程設計經驗。在1993至2000年他擔任初始 D2 Technologies 的共同合夥人及技術長，致力於設計及發展公司在語音處理方面的技術及架構。在2002年 D2 Technologies 被合併的期間擔任 GlobespanVirata 的首席工程師，監督 ATM 語音技術及 IP 語音軟體技術，之後再合夥創辦 D2 Technologies 及 D2 Nova ，擔任技術長。 Lindsay 最一開始是 AT&T Bell 實驗室的技術成員，之後改在 Digital Sound Corporation 擔任資深工程師，而後又成為 Spectron MicroSystems 的研發副總，是首個 DSP 作業系統的革新者。 Lindsay 擁有許多語音處理相關技術的專利及學位，包含密西根大學電機系的碩士及理海大學的電機系學士。

Jon Wu

總經理

吳炯憲博士在軟體開發的設計及管理上擁有超過20年的專業經驗。原本創立的 Webia Technologies 與 D2 Technologies 於2011年合併後，便加入 D2 家族，在那之前也曾是 INVENTEC 的網路業務部門首長。吳博士所帶領的團隊曾經為許多知名大廠像是 Cisco,Netgear 及 Skype 等研發 VoIP 產品。在2000年之前，吳博士曾在工研院的電腦通訊研究中心工作，負責發展 3GPP RLC/MAC 技術以及行動多媒體應用，並且在行動網路相關技術上獲得超過10項作者專利以及卓越的研究貢獻獎項。吳博士擁有台灣大學資訊工程系學士及清華大學資訊工程研究所博士學位。

Matt Randmaa

開發維運技術副總

Matt Randmaa 擁有超過20年的專業經驗在信號處理及 IP 通信軟體技術的發展及設計上。致力於設計並優化信號處理軟體在 IP 語音上的應用相關技術，包含回音消除、語音壓縮、封包遺失補償、抖動緩衝器及傳真、來電顯示等。在2003年成為 D2 Technologies 共同創辦合夥人之前，曾是 GlobespanVirata 的軟體工程總監。在網路語音技術相關主題上是權威性的代表，並參與許多 VoIP 相關的專利研發，有加州大學聖塔芭芭拉分校電機系的碩士學歷。
