EVOX 致力打造專業靈活的企業通訊平台，整合各種渠道的雲端服務，助企業拓展商機、創造顧客服務新體驗。

節能減碳

EVOX 運行於 AWS，可有效降低 90% 碳排放

以往公司若使用一台 350W 中小型 IP PBX，所使用的總機設備、100 台 IP 電話機再加上 3 坪冷氣機房，一年大約會產生 26,718 度的電，相當於 13.6 公噸 CO2e，若改用 EVOX 雲端通訊平台，一年則只會排放 1.36 公噸 CO2e。

全球化經營

EVOX 運行於公有雲，可輕易滿足行動分機、遠距辦公 WFH、全國或全球多據點等需求

  • 辦公地點不再受到居住地、疫情或天災人禍等因素的限制
  • 員工可以視需求選擇適合的裝置來使用
  • IT 或總務可遠距管理電話系統
  • 快速建立多據點、分公司、新門市、海外據點

資訊安全

資訊安全與防護

  • EVOX 核心網位於 AWS，受 SBC 保護
  • EVOX App 以 OAuth 2.0 和 2-factor authentication 做身分驗證

資安認證

  • 通過台灣工業局 MAS 行動資安聯盟認證
  • 通過弱點掃描與滲透測試

個資保護

  • 遵循歐洲 GDPR 與台灣個資保護法

企業防詐騙

  • EVOX 與 Whoscall 深度整合，保護員工不被惡意來電騷擾
